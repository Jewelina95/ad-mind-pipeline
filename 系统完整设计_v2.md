# 3. 合成数据生成器 v2 + 分布对标

## 3.1 v1 vs v2

| | v1 (现有) | v2 (重做) |
|---|---|---|
| 文件路径 | `data/baseline 4.8/generate_synthetic_ad.py` | `data/baseline 4.8/generate_synthetic_temporal.py` (新建) |
| 输出粒度 | 1 个任务 5-10 分钟 CSV | 30 天每天多任务 + EMA + 量表 |
| 文件数 | 84 个 (4 人 × 3 期 × 7 任务) | 20 个虚拟患者 × 30 天数据 |
| 退化建模 | 离散 3 档 (MCI/mild/moderate) | 连续光谱 + 5 种进展模式 |
| 数据源 | 仅 4 人 baseline + 文献效应量 | + WearGait-PD 分布 + ds004504 MMSE + 临床 xlsx |
| 多模态 | 仅 sensor | sensor + EMA + 量表 + clinical note |

## 3.2 v2 设计原则（5 条）

1. **纵向不横断**: 每个虚拟患者是 30 天连续轨迹
2. **进展光谱**: progression ∈ [0, 1]，不直接贴 stage 标签
3. **分布对齐**: 退化幅度参考真实数据集实测，不只是文献效应量
4. **多模态对齐**: sensor 异常 ↔ EMA mood 下降 ↔ 量表分降低 ↔ note 描述同步
5. **个性化**: 每人有 persona (年龄/教育/进展模式)

## 3.3 ⭐ 分布对标具体怎么做

这一步是"让你合成的患者真的像真实患者"的关键。

### 3.3.1 问题：你 v1 的退化系数是猜的

```python
# 你 generate_synthetic_ad.py 现状
'walking_normal': {
    'speed_factor': [0.85, 0.92],   # ← 这个 0.85 是哪来的？文献？拍的？
    'stride_jitter': [0.03, 0.06],  # ← 同上
}
```

**分布对标 = 拿真实患者数据测量出真值，替换这些猜的数。**

### 3.3.2 4 步操作（用 WearGait-PD 举例）

#### Step 1: 下载并提取真实患者特征

#### Step 3: 生成器读这个 JSON 而不是硬编码

```python
# generate_synthetic_temporal.py (修改后)
DISTRIBUTIONS = json.load(open("data/external/distributions.json"))

def degrade_gait_speed(baseline_value, progression):
    """progression ∈ [0,1], 0=健康, 1=mild_AD"""
    target_at_severe = DISTRIBUTIONS["imu_walking_normal"]["gait_speed"]["degradation_at_severe"]
    # 例 target=0.78, progression=0.5 → 退化系数 = 1 - (1-0.78)*0.5 = 0.89
    factor = 1 - (1 - target_at_severe) * progression
    return baseline_value * factor + np.random.normal(0, 0.03)
```

**关键差别**：
- 之前: `speed_factor=[0.85, 0.92]` （拍的）
- 现在: `factor=0.89` （从 100 个真实 PD 患者算的）

#### Step 4: 验证你合成数据的分布真的像真实患者

```python
# scripts/validate_synthetic_distribution.py
from scipy import stats

# 读 你的合成 mild_AD 患者步速
synth_speeds = [...]  # 从 data/synthetic_v2/P01..P20 抽 progression≈1 的天数

# 读 WearGait-PD 真实 PD 步速
real_speeds = pd_df["gait_speed"].values

# Kolmogorov-Smirnov 检验：两个分布是不是一致
ks_stat, p_value = stats.ks_2samp(synth_speeds, real_speeds)
print(f"KS test: stat={ks_stat:.3f}, p={p_value:.3f}")
# p > 0.05 → 你合成的分布和真实分布"统计上没差异" ← 对标成功
# p < 0.05 → 差距大, 回去调系数
```

**论文里可以写**:
> "Synthetic mild-AD trajectories were validated against WearGait-PD distributions; Kolmogorov-Smirnov test showed no significant difference in gait speed (p=0.34) and stride variability (p=0.18)."

## 3.4 v2 生成器代码骨架

```python
# data/baseline 4.8/generate_synthetic_temporal.py

import numpy as np
import pandas as pd
import json
from pathlib import Path
from dataclasses import dataclass

# ── 加载真实分布参考 ──
DISTRIBUTIONS = json.load(open("data/external/distributions.json"))

# ── Persona 定义 ──────────────────────────────────────
PROGRESSION_PATTERNS = {
    "linear":      lambda d: d / 30,
    "stepwise":    lambda d: 0 if d < 14 else 0.5 if d < 21 else 0.8,
    "plateau":     lambda d: min(d / 7, 0.4),
    "fluctuation": lambda d: 0.3 + 0.2*np.sin(d/3),
    "acute_event": lambda d: 0 if d < 20 else 0.7,
}

PERSONAS = [
    {"id": "P01", "base": "zewei",  "age": 68, "edu": 12, "pattern": "linear"},
    {"id": "P02", "base": "zewei",  "age": 73, "edu": 6,  "pattern": "stepwise"},
    {"id": "P03", "base": "junkai", "age": 70, "edu": 16, "pattern": "plateau"},
    {"id": "P04", "base": "junkai", "age": 75, "edu": 9,  "pattern": "fluctuation"},
    {"id": "P05", "base": "jialu",  "age": 72, "edu": 12, "pattern": "acute_event"},
    # ... 共 20 个
]

# ── 退化映射（参考分布对标后的真实数据）──
def degrade_value(baseline, progression, modality, feature):
    target = DISTRIBUTIONS[modality][feature]["degradation_at_severe"]
    factor = 1 - (1 - target) * progression
    return baseline * factor

# ── 量表映射 (从 ds004504 + 临床 xlsx 拟合) ───────────
def progression_to_mmse(p, age, edu):
    """progression ∈ [0,1] → MMSE 分数
    拟合自 ds004504: AD μ=17.75 σ=4.5, CN μ=30
    """
    base = 30 - 12 * p
    base += np.random.normal(0, 1.5)
    base += 0.05 * (edu - 12)  # 教育修正
    return np.clip(base, 0, 30)

def progression_to_phq9(p):
    return np.clip(np.random.normal(2 + 8*p, 2), 0, 27)

# ── 生成器主体 ────────────────────────────────────────
@dataclass
class DailyRecord:
    day: int
    progression: float
    sensor_data: dict[str, pd.DataFrame]
    ema: list[dict]
    survey: dict | None
    note: str | None

class TrajectoryGenerator:
    def __init__(self, persona, baseline_data, kb_store):
        self.persona = persona
        self.baseline = baseline_data
        self.kb = kb_store
        self.progression_fn = PROGRESSION_PATTERNS[persona["pattern"]]
    
    def generate_30_days(self) -> list[DailyRecord]:
        records = []
        for day in range(30):
            p = self.progression_fn(day)
            
            sensor = {}
            for task in ["walking_normal", "walking_dual_task", 
                         "balance_standing", "hand_fine_motor"]:
                sensor[task] = self._degrade_task(task, p)
            
            ema = [self._gen_ema(day, h, p) for h in [9, 14, 20]]
            
            survey = self._gen_survey(day, p) if day % 7 == 0 else None
            note = self._gen_note(day, p, records) if day == 14 else None
            
            records.append(DailyRecord(day, p, sensor, ema, survey, note))
        return records
    
    def _degrade_task(self, task, p):
        base = self.baseline[task].copy()
        for col in ["svm_std", "jerk_std", "hr_std", "gsr_cv_pct"]:
            if col in base.columns:
                base[col] = degrade_value(base[col], p, "imu_walking_normal", col)
        base += np.random.normal(0, 0.05, base.shape)
        return base
    
    def _gen_ema(self, day, hour, p):
        return {
            "day": day, "hour": hour,
            "mood": int(np.clip(np.random.normal(7-4*p, 1), 1, 10)),
            "sleep_quality": int(np.clip(np.random.normal(7-3*p, 1), 1, 10)),
            "anxiety": int(np.clip(np.random.normal(2+5*p, 1), 0, 10)),
        }
    
    def _gen_survey(self, day, p):
        return {
            "day": day,
            "phq9": progression_to_phq9(p),
            "moca_est": progression_to_mmse(p, self.persona["age"], self.persona["edu"]),
        }
    
    def _gen_note(self, day, p, history):
        templates = self.kb.get_note_templates()
        return self._fill_template(templates, p, history)


def main():
    out_dir = Path("data/synthetic_v2")
    out_dir.mkdir(exist_ok=True)
    
    baseline_data = load_all_baselines()
    kb = KnowledgeStore()
    
    for persona in PERSONAS:
        gen = TrajectoryGenerator(persona, baseline_data[persona["base"]], kb)
        trajectory = gen.generate_30_days()
        save_trajectory(out_dir / persona["id"], trajectory)

if __name__ == "__main__":
    main()
```

# 4. ANALYZER LAYER

## 4.1 角色

把"原始 sensor 数据 + EMA + 量表"转成"医生能看懂的事实清单"。**不调 LLM 做这一步**——纯统计 + 模板。

## 4.2 5 类 Fact 的检测公式

### Outlier (异常值)
```python
def detect_outlier(current, baseline_mean, baseline_sd, threshold=2.0):
    z = (current - baseline_mean) / baseline_sd
    if abs(z) > threshold:
        return DataFact(
            type="outlier",
            template_text=f"{entity} 异常 {('升高' if z>0 else '下降')}至 {current:.2f} "
                         f"(基线 {baseline_mean:.2f}±{baseline_sd:.2f}, z={z:+.1f})",
            severity=abs(z) / threshold,
        )
```

### Trend (趋势)
```python
from scipy.stats import linregress

def detect_trend(time_series, min_days=7):
    if len(time_series) < min_days:
        return None
    days, values = zip(*time_series)
    slope, _, r, p, _ = linregress(days, values)
    if p < 0.05 and abs(r) > 0.3:
        direction = "下降" if slope < 0 else "上升"
        return DataFact(
            type="trend",
            template_text=f"{entity} 在过去 {len(days)} 天呈 {direction} 趋势 "
                         f"(从 {values[0]:.2f} 到 {values[-1]:.2f}, slope={slope:+.3f}/天)",
        )
```

### Comparison (对比)
```python
def detect_comparison(window1_values, window2_values):
    m1, m2 = np.mean(window1_values), np.mean(window2_values)
    pct = (m2 - m1) / m1 * 100
    if abs(pct) > 10:
        return DataFact(
            type="comparison",
            template_text=f"{entity} 均值由 {m1:.2f} 变化为 {m2:.2f} ({pct:+.1f}%)",
        )
```

### Difference (任务差异)
```python
def detect_dual_task_cost(single_value, dual_value):
    dtc = (single_value - dual_value) / single_value * 100
    if dtc > 20:
        return DataFact(
            type="difference",
            template_text=f"双任务代价 {dtc:.1f}% "
                         f"(单任务 {single_value:.2f}, 双任务 {dual_value:.2f})",
        )
```

### Extreme (极值)
```python
def detect_extreme(window_values, baseline_max, baseline_min):
    if max(window_values) > baseline_max * 1.2:
        return DataFact(
            type="extreme",
            template_text=f"{entity} 达到近期极大值 {max(window_values):.2f} "
                         f"(超基线上限 {baseline_max:.2f})",
        )
```

## 4.3 完整 Analyzer 代码骨架

```python
# src/skills/analyzer.py

from pathlib import Path
import pandas as pd
import numpy as np
from scipy.stats import linregress
from dataclasses import dataclass
from typing import Literal

@dataclass
class DataFact:
    fact_type: Literal["outlier", "trend", "comparison", "difference", "extreme"]
    entity: str
    modality: str
    template_text: str
    severity: float
    raw_data_ref: str
    timestamp: str

class Analyzer:
    """Discoverer + Inquirer + Planner 三合一"""
    
    def __init__(self, baseline_csv: Path, llm_client=None):
        self.baseline = pd.read_csv(baseline_csv).set_index("task")
        self.llm = llm_client
        
    def discover(self, patient_records: list[dict]) -> list[DataFact]:
        facts = []
        
        for task in ["walking_normal", "balance_standing", "hand_fine_motor"]:
            for feat in ["svm_std", "jerk_std", "hr_std", "gsr_cv_pct"]:
                series = self._extract_feature(patient_records, task, feat)
                if len(series) == 0:
                    continue
                
                base_mean = self.baseline.loc[task, f"{feat}_mean"]
                base_sd = self.baseline.loc[task, f"{feat}_sd"]
                latest_day, latest_val = series[-1]
                
                outlier = self._detect_outlier(feat, latest_val, base_mean, base_sd, latest_day)
                if outlier: facts.append(outlier)
                
                trend = self._detect_trend(feat, series)
                if trend: facts.append(trend)
                
                comp = self._detect_comparison(feat, series)
                if comp: facts.append(comp)
        
        dtc = self._detect_dual_task_cost(patient_records)
        if dtc: facts.append(dtc)
        
        facts.extend(self._analyze_ema(patient_records))
        facts.extend(self._analyze_surveys(patient_records))
        return facts
    
    def inquire(self, agent_role: str, facts: list[DataFact]) -> list[str]:
        prompt = f"""你是一位{agent_role}。看到以下事实：
{chr(10).join(f.template_text for f in facts)}

作为这个领域的专家，你会进一步追问哪 3 个问题？只列问题，每条一行。"""
        response = self.llm.messages.create(
            model="claude-sonnet-4-20250514",
            max_tokens=512,
            messages=[{"role": "user", "content": prompt}],
        )
        return [line.strip() for line in response.content[0].text.split("\n") if line.strip()]
    
    def plan(self, facts: list[DataFact]) -> dict[str, list[DataFact]]:
        routing = {"physio": [], "behavior": [], "clinical": []}
        for f in facts:
            if f.modality in ("imu", "ppg", "audio"):
                routing["physio"].append(f)
            elif f.modality == "eda" or "ema" in f.entity:
                routing["behavior"].append(f)
            if "survey" in f.entity or "stage" in f.entity:
                routing["clinical"].append(f)
        return routing
```

## 8.2 整个流程时序图

```
T+0ms    [无 LLM] 读 records / patient / context        500ms
T+500ms  [无 LLM] Analyzer.discover() 算 15 facts       2s
T+2.5s   [无 LLM] Analyzer.plan() 路由                  10ms
T+2.5s   ┌────[LLM 1] PhysioAgent       ─┐ 并发        4s
         ├────[LLM 2] BehaviorAgent     ─┤              4s
         └────[LLM 3] ClinicalAgent     ─┘             (依赖前两位, 串行)
T+10s    [LLM 4] Narrator 抛光叙事                       3s
T+13s    [无 LLM] 拼 JSON, 完成

总计: ~13 秒 (含 4 次 LLM 调用)
Token 消耗: 约 6000 input + 2000 output ≈ $0.03 / 次
```

## 8.3 关键决策：哪几次"喂"是真喂

| 步骤 | 输入 | LLM | 是不是"喂给 Agent" |
|---|---|---|---|
| Step 1 | 文件→内存 | ✗ | 不是 |
| Step 2 | records→Analyzer | ✗ | 不是 |
| Step 3 | facts→routing | ✗ | 不是 |
| **Step 4** | facts+KB+context | **✓ 第一次喂** | **★ 喂给 PhysioAgent** |
| **Step 5** | facts+KB+context | **✓ 第二次喂** | **★ 喂给 BehaviorAgent** |
| **Step 6** | Insights+KB+量表 | **✓ 第三次喂** | **★ 喂给 ClinicalAgent** |
| Step 7 | all_insights | ✓ 抛光 | 喂给 Narrator |

**"喂"= 把数据写进 prompt 发给 Claude API**。这就是"agent 怎么喂"的具体动作。

## 8.4 总结要记住的 5 件事

1. **Agent 不是预先训出来的**——每次有新患者数据，从头跑一次 Step 1~7
2. **同一份 KB 反复用**——每次评估都重新 query, 不缓存到 Agent 状态
3. **同一份 baseline 反复用**——每次都和它对比
4. **每个 Agent 一次 LLM 调用**——不是循环、不是迭代
5. **3 个 Agent 串/并不固定**：PhysioAgent + BehaviorAgent 可并发，ClinicalAgent 依赖前两位所以串行

# 9. 论文里怎么写这个架构

```latex
\subsection{System Architecture}

We extend the MIND framework~\cite{zou2026mind} with three modifications 
tailored to AD wearable monitoring:

(1) \textbf{Multi-modal sensor input}: Beyond MIND's passive smartphone 
sensing, we integrate continuous EDA, PPG, IMU, and microphone signals 
from a wearable glove.

(2) \textbf{Multi-agent synthesizer}: Rather than MIND's single LLM 
synthesis under the biopsychosocial framework, we decompose synthesis 
into three specialty agents (Physio / Behavior / Clinical) inspired by 
clinical MDT practice~\cite{mac2025}, each grounded in a curated knowledge 
base of AD-specific evidence.

(3) \textbf{Closed-loop intervention}: Whereas MIND focuses on 
retrospective clinician review, our ClinicalAgent triggers real-time 
non-pharmacological interventions (music therapy, breathing guidance) 
when BehaviorAgent detects pre-agitation patterns.

The Analyzer-Synthesizer-Narrator pipeline preserves MIND's modularity 
while extending it from clinician-facing dashboards to a full 
monitoring-intervention closed loop.
```

# 附录 B: 关键概念速查表

| 你以为 | 实际是 |
|---|---|
| 训练 Agent | Agent 不训练，每次跑都重新读输入 |
| MIND 用了 ML | MIND 0 个训练模型，纯 rule + LLM |
| 我合成数据是给 Agent 学习用的 | 是给 demo / 验证 / 论文用的 |
| 7 Agent 全部保留 | 砍到 3 个，反而清爽 |
| AI 替医生诊断 | AI 把数据呈现得医生看得懂 |

| 名词 | 含义 | 在你系统里 |
|---|---|---|
| ML | 用数据训分类器，输出 .pkl | **不用** |
| LLM | Claude/GPT, 已训好, prompt 调用 | 用，3 次/评估 |
| Rule-based | Python 写公式，不训练 | Analyzer 全部是这种 |
| Fact | 一句话事实，统计算出来的 | Analyzer 输出 |
| Insight | 结合 KB 推理后的临床观察+含义 | Agent 输出 |
| Dashboard | 叙事+图表组合，给医生看 | Narrator 输出 |
| 分布对标 | 用真实数据集校准合成数据 | 生成器 v2 关键 |
